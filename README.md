# DeepSeek Prompt Keeper

![图标](icons/icon128.png) (请替换为实际图标)

在 DeepSeek 聊天页面中高效管理你的提示词，无需再依赖外部剪切板。支持快捷键、置顶、拖拽排序和导入导出，所有数据均存储在本地，保护隐私安全。

---

## ✨ 功能特性

- **📋 提示词管理**  
  在侧边栏中保存、编辑、删除你的常用提示词，一键插入聊天输入框。

- **⌨️ 快捷键快速插入**  
  `Alt + 1` ~ `Alt + 0` 直接插入前 10 个提示词（按当前显示顺序），无需打开侧边栏。

- **📌 置顶功能**  
  点击提示词左侧的星标（☆/★）可将常用提示词置顶，置顶项始终显示在列表顶部。

- **↕️ 拖拽排序**  
  按住提示词左侧的 `⋮⋮` 手柄即可在同组内拖动排序，跨组拖拽时会提示使用星标切换。

- **📥📤 导入/导出**  
  将提示词导出为 JSON 文件备份，或从备份文件恢复，数据迁移轻松无忧。

- **🔒 本地存储，隐私安全**  
  所有提示词仅保存在浏览器的 IndexedDB 中，不会上传任何服务器，清除浏览器数据前请务必导出备份。

- **🚀 轻量无侵入**  
  以浏览器扩展形式运行，不修改 DeepSeek 官方页面源码，仅注入必要 UI 元素。

## 📦 安装方法

1. 下载本仓库源码并解压，或 `git clone`。
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`。
3. 开启右上角的“开发者模式”。
4. 点击“加载已解压的扩展程序”，选择包含 `manifest.json` 的文件夹。
5. 扩展加载成功后，访问 DeepSeek 聊天页面即可使用。

---

## 🎯 使用方法

### 基本操作
1. 点击页面右侧的 📋 按钮，打开提示词库侧边栏。
2. 点击 **+ 新增** 按钮，输入标题和内容，保存当前输入框中的文本（或自定义内容）。
3. 点击任意提示词卡片，可选择替换或追加到输入框。

### 置顶与排序
- **置顶**：点击提示词左侧的 ☆/★ 按钮切换置顶状态，置顶项会显示在列表最上方。
- **拖拽排序**：按住 `⋮⋮` 手柄，在同组（置顶组或普通组）内上下拖动调整顺序。跨组拖动（如将置顶项拖到普通组）会被拦截，请使用星标切换组。

### 快捷键
- `Alt + 1` ~ `Alt + 9`：插入列表中第 1 ~ 9 个提示词。
- `Alt + 0`：插入列表中第 10 个提示词。
> 快捷键在页面刷新后立即可用，无需先打开侧边栏。

### 导入/导出
- **导出**：点击 **📤 导出** 按钮，将当前所有提示词保存为 JSON 文件。
- **导入**：点击 **📥 导入** 按钮，选择之前导出的 JSON 文件，即可恢复提示词（重复导入会添加新记录，不会覆盖原有数据）。

## ⚠️ 注意事项

- **数据安全**：所有数据仅保存在本地，清除浏览器缓存或使用无痕模式可能导致数据丢失。请定期导出备份。
- **输入框识别**：扩展会自动查找 DeepSeek 的输入框，若因页面改版无法找到，可点击底部警告中的“手动选择”，然后点击页面上的输入框。
- **快捷键冲突**：如果 `Alt+数字` 与浏览器或其他扩展冲突，可尝试修改代码中的快捷键绑定（需自行修改 `content.js` 中的 `setupKeyboardShortcuts` 函数）。
- **隐私声明**：本扩展不收集任何用户数据，仅在用户本地存储提示词。完整隐私政策可查看 [此处]()（待补充）。
